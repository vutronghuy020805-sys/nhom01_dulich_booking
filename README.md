## 3. Mô tả dự án
VieGo là nền tảng booking du lịch với các chức năng:
- Tìm kiếm dịch vụ du lịch
- Xem chi tiết dịch vụ
- Đặt chỗ và thanh toán
- Vé xe khách
- Thuê xe
- Đưa đón sân bay
- Hoạt động du lịch
- Biệt thự / Căn hộ
- Blog du lịch
- Hỗ trợ khách hàng

## 4. Tính năng nổi bật
- Giao diện responsive
- Hệ thống điều hướng nhiều trang
- Flow đặt dịch vụ hoàn chỉnh
- Blog du lịch
- Dark mode
- Chatbot hỗ trợ cơ bản
- Deploy bằng GitHub Pages
